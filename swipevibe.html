Step 1: Decide

Use the main buttons to Like (❤️), Skip (❌), or give a Priority Like (⭐) to a profile.

Step 2: Revert

Made a mistake? Hit the Revert (🔄) button to get the last profile back. You can only do this once per swipe!

Step 3: Match!

If you both Like each other, it’s a Match! A popup will appear, and you can start a conversation.
